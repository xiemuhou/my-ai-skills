# pptx — 教程类 PPTX 制作 Skill

`pptx` 用来把主题、资料、笔记或大纲制作成教程类 PowerPoint。它默认使用简洁现代、明亮浅色、技术课程网页感的教程模板，适合课程、培训、操作指南和技术分享。

## 快速开始

```text
请使用 pptx skill 制作一个 10 页教程类 PPT
主题：如何使用 Codex Skills
受众：刚开始使用 Codex 的开发者
风格：简洁现代，明亮主题，技术感
输出：pptx 文件
```

## 适合场景

| 你的需求 | 推荐说法 |
| --- | --- |
| 做教程型 PPT | `做一个教程类 PPT，主题是...` |
| 做课程课件 | `生成一份课件，面向初学者` |
| 做操作指南 | `把这些步骤整理成培训 PPT` |
| 做知识讲解 | `用 PPT 解释这个概念，偏教程风` |
| 做技术分享 | `生成一份技术分享 PPT，包含代码页和实践建议` |
| 改造已有大纲 | `把这个大纲做成 pptx，使用教程模板` |

## 默认风格

| 项目 | 规范 |
| --- | --- |
| 主题 | 明亮主题 |
| 主色 | 深绿色或科技蓝 |
| 强调色 | 青绿色、浅蓝色 |
| 背景 | 浅灰蓝到白色的柔和渐变，可带很淡径向光晕 |
| 气质 | 简洁、现代、清爽、技术感、教学感 |
| 结构 | 标题、学习目标、步骤、示例、代码、误区、复盘 |
| 适用 | 教程、课件、内部培训、方法论讲解、技术分享 |

## 品牌 Logo

默认提供一个可复用 Logo：

[assets/xiemuhou-wordmark-logo.svg](./assets/xiemuhou-wordmark-logo.svg)

Logo 以 `xiemuhou` 为主字标，搭配 `blog.xmhweb.cn`，不带外边框，默认放在页面左下角。生成 PPT 时应自然融入版式，保持低干扰、高识别度。

使用建议：

- 所有页面使用同一 Logo 尺寸，不区分封面页和正文页。
- Logo 默认宽度为页面宽度的 12%，固定左下角。
- Logo 应作为背景品牌层或母版层处理，不作为正文内容参与排版。
- 正文、页码、代码块、表格、卡片或图表不能进入 Logo 安全区。
- 如果你提供 `--logo <path/url>`，优先使用你指定的 Logo。

## 默认页面结构

如果用户没有指定大纲，默认生成：

```text
1. 封面
2. 为什么要学
3. 学习地图
4. 核心概念
5. 操作步骤
6. 代码或示例演示
7. 方案对比
8. 常见误区
9. 检查清单
10. 总结与下一步
```

## 代码页规范

当 PPT 里需要放代码时：

- 使用等宽字体，例如 `Consolas`、`Cascadia Code`、`JetBrains Mono`。
- 每页代码不超过 12 行。
- 保留关键片段，不把完整文件塞进一页。
- 使用语法高亮区分关键字、字符串、注释和函数名。
- 用标注解释关键行。

## Prompt 示例

基础教程：

```text
请使用 pptx skill 制作 10 页教程类 PPT
主题：Python 虚拟环境入门
受众：没有工程经验的初学者
风格：简洁现代，明亮主题，技术感
```

操作指南：

```text
请使用 pptx skill 把下面步骤做成培训 PPT
主题：如何安装和更新 my-ai-skills
要求：每页一个步骤，最后加检查清单
```

技术分享：

```text
请使用 pptx skill 生成 12 页技术分享 PPT
主题：如何设计高质量 Agent Skill
难度：intermediate
要求：包含代码示例、反例、最佳实践和课后练习
```

指定输出路径：

```text
请使用 pptx skill 制作教程类 PPT
主题：SNMP 协议配置入门
--pages 10
--audience 网络工程师
--output D:\slides\snmp-tutorial.pptx
```

## 可选参数

| 参数 | 作用 |
| --- | --- |
| `--pages <n>` | 指定页数 |
| `--audience <text>` | 指定受众 |
| `--level beginner|intermediate|advanced` | 指定难度 |
| `--style tutorial|workshop|course|guide|tech-share` | 指定教程子风格 |
| `--theme light|dark` | 指定明亮或深色主题，默认 `light` |
| `--logo <path/url>` | 指定 Logo |
| `--brand xmh|xiemuhou|谢幕后` | 指定默认品牌文字 |
| `--output <path>` | 指定输出路径 |
| `--language zh|en` | 指定语言 |

## 输出

默认输出一个 `.pptx` 文件，并在完成后说明：

- 文件路径
- 页数
- 使用的模板风格
- 是否完成基础校验

## 注意事项

- 如果你只给主题，AI 会自动补齐大纲。
- 如果你给了资料，AI 会先重构成教学路径，再制作 PPT。
- 这个 skill 偏教程和培训，不适合做夸张营销风首页。
- 生成 PPTX 时应尽量做可视化检查，避免文字溢出或元素重叠。
