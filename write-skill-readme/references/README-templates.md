## 模板使用指南

### 何时使用模板 A（功能型）

适用条件：
- 技能主要通过 Prompt 触发
- 有明确的工作流程
- 需要解释设计理念/原理
- 有多个使用场景需要展示

典型案例：
- systematic-literature-review
- get-review-theme
- nsfc-*-writer 系列技能

### 何时使用模板 B（工具型）

适用条件：
- 技能主要通过脚本/命令行调用
- 功能相对简单直接
- 不需要复杂的使用场景解释
- 用户主要是开发者

典型案例：
- install-bensz-skills
- init-project
- 工具类技能

### 何时使用模板 C（混合型）

适用条件：
- 技能支持 Prompt 和脚本两种方式
- 推荐 Prompt 调用，但脚本作为备选
- 需要详细说明脚本步骤
- 有独立的 `scripts/` 目录

典型案例：
- make-latex-model
- 技能类工具
